== Description ==

Easily integrate Mailchimp with your multi-site or regular wp site.

What it does?
The plugin Mailchimp Integration will syncronize all existing users on your install.

Note: After plugin installation you need to visit plugin settings from the WP Admin go to Settings -> Mailchimp and enter your Mailchimp API Key.

== Installation ==

1. Connect to your website using FTP 
2. Include the "mailchimp-integration" folder into /wp-content/plugins/
3. Activate Mailchimp Integration from the WP Admin.
4. Visit plugin settings page -> Settings -> Mailchimp and enter your Mailchimp API Key.
